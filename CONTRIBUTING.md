# Contributing

Thank you for your interest in Intelligence Engine!

This repository is a **read-only showcase** of the project's capabilities.
The source code is proprietary and not available for contributions.

## What This Repository Contains

- Documentation and architecture overviews
- Interactive demos demonstrating key features
- Screenshots and diagrams

## What This Repository Does NOT Contain

- Source code
- Implementation details
- Build scripts

## Questions?

For questions about the project, please open an issue or contact the maintainer.
